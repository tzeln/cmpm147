Experiment 3 - Alternative Worlds

Description

In this experiment, I'm aiming to create randomly generated tiled worlds.

Technical

By mapping a grid filled with symbols, I can correspond them to a grid holding actual sprite values. With some additional tweaking on autotiling logic and altering sprites based on surrounding contexts, the goal was to create randomly generated worlds that made sense every single time. As you can see this didn't work out.

Reflection

I was a little ambitious with the room generation and that's why it looks bad. I simply am not willing to sacrifice anymore time putting work into the assignment when I have other work to do.
